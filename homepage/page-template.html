새 페이지 템플릿

앞으로는 새 페이지 만들 때 공통 추적코드, 헤더, 푸터를 따로 기억할 필요 없습니다. 아래에서 필요한 템플릿만 복사하면 됩니다.

1. 일반 페이지

메인, 코칭, 커리큘럼처럼 상단 헤더와 하단 푸터가 같이 필요한 페이지입니다.

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>페이지 제목</title>
</head>
<body>
  페이지 본문

  <script src="homepage-shell.js?v=d5608a0"></script>
</body>
</html>

2. 결제용 단독 페이지

카드결제 안내처럼 헤더/푸터는 없이, GA4와 Clarity만 붙이고 싶은 페이지입니다.

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>결제용 페이지 제목</title>
</head>
<body>
  페이지 본문

  <script src="homepage-shell.js?v=d5608a0" data-header="false" data-footer="false"></script>
</body>
</html>

3. 실제 동작

homepage-shell.js 가 공통으로 세 가지를 처리합니다. GA4/Clarity 추적, 공통 헤더, 공통 푸터입니다. 결제용 페이지는 옵션으로 헤더/푸터만 끌 수 있습니다.
