1. 아래 카탈로그에서 원하는 슬라이드를 클릭해서 열기
2. E키를 누르면 편집 모드 (드래그로 이동, 텍스트 클릭해서 수정, 이미지 Ctrl+V로 추가)
3. E키를 다시 누르면 자동 저장 (GitHub에 직접 커밋)
4. 1~2분 후 반영됨 (Vercel CDN 배포)
