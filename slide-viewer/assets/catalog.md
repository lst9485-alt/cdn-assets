# 에셋 카탈로그

에셋 추가/삭제 시 이 파일도 함께 업데이트.

## JS/CSS

| 파일 | 설명 | 사용처 |
|------|------|--------|
| slide-style.css | 공용 CSS (정본) | 에디터 + 생성 슬라이드 모두 외부 참조 |
| slide-editor.js | 에디터 JS (build-editor.py가 생성) | 에디터 3개 파일에서 외부 참조 |
| slide-engine.js | 생성 슬라이드 전용 JS | 생성된 HTML에서 사용 |
