상담 준비 (8단계)
1고객 확인
이름/전화번호 필터 또는 오늘/내일 자동 조회
↓
2사전질문지 읽기
CRM 스프레드시트에서 고객 응답 확인
↓
3세그먼트 판별
A/B/C/D 고객 타입 분류
↓
4레퍼런스 참고
상담목적별 패턴 읽기 (과거 유사 케이스)
↓
5팩트 조사
정책대출 자격 / 세금 / LTV / 대출조건 조사
↓
6맞춤 답변 생성
신청계기/기본정보/재무상황 기반 답변 작성
↓
7Notion DB 반영
상담 DB에 준비 데이터 입력
↓
8준비 완료
상담 대기 상태
상담 후처리 (5단계)
1고객 확인
Notion DB에서 오늘 상담 자동 조회
↓
2회의록 읽기
Google Docs 상담 기록 확인
↓
3핵심 요약 생성
핵심사항 / 추진방향 요약
↓
4Google Docs 반영
요약본 Docs에 추가
↓
5피드백 + 보고
사용자 확인 및 최종 정리
