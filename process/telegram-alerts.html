리마인더 그룹에 실제로 들어오는 알림 기준으로 정리한 흐름

태스크 관리 (3가지)
1하루 4회 할 일 받기
→ 08:00 / 14:00 / 19:00 / 22:00 리마인더 확인
자동: 오늘 할 일, 지연 항목, 내일 할 일, 완료 버튼을 묶어서 발송
소스: 01-2 태스크 리마인더
2버튼이나 명령어로 바로 처리하기
→ ✅ 버튼 또는 오늘 / 목록 / 추가 / 완료 / 변경 입력
자동: Notion 태스크 완료, 조회, 생성, 일정 변경까지 반영
소스: 01-3 태스크 완료, 01-4 태스크 명령어
3시간 맞춰 개별 알림 받기
→ 정해진 시각에 뜨는 단건 알림 보고 바로 처리
자동: 시간 포함 태스크만 ±2분 범위로 잡아서 한 건씩 발송
소스: 01-5 시간별 리마인더
아침 플래너 (1가지)
4하루 시작 전에 플랜 확인하기
→ 원씽 / P2 / P3 / 오늘 마감 태스크를 한 번에 확인
자동: 어제 루틴 미기입, 루틴 연속일, 주간 평균까지 함께 묶어서 발송
소스: nightwatch morning-planner
시스템 · 예외 (2가지)
5n8n 상태 이상 확인하기
→ 중단 / 복구 알림이 오면 바로 상태 확인
자동: 컨테이너 상태가 바뀐 순간에만 알림 보내기
소스: tools/scripts/n8n-monitor.sh
6동기화 오류 확인하기
→ Notion → Obsidian 동기화 실패 알림 보고 수동 점검
자동: 실패 시 즉시 발송. 같은 방으로 오지만 발신 credential은 우리동네봇인 예외
소스: 02 notion 콘텐츠캘린더→obsidian
